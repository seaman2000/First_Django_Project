# ⚽ Match Tracker (Django Project)

This is my first project built with **Django**.  
It is a simple web application for tracking football matches and their odds.

## 📌 Description

Match Tracker allows users to add and manage information about matches, including:

- 🏠 Home team  
- 🚗 Away team  
- 💰 Odds  
- 📅 Match date and time  

The main goal of this project is to practice Django fundamentals such as models, views, forms, and templates.

---

## ⚙️ Features

- ➕ Add a new match  
- 📋 Display matches *(optional, if implemented)*  
- 🧾 Store data in a database  
- 📆 Input date and time
